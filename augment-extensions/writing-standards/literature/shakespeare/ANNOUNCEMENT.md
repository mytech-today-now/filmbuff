# 🎭 Shakespeare Writing Standards Module - Now Available!

## Release Date
February 24, 2026

## Overview

We're thrilled to announce the release of the **Shakespeare Writing Standards** module for Augment Extensions! This comprehensive module provides AI-driven guidance for writing in the style of William Shakespeare, covering poetry, drama, language, character development, and thematic elements.

## What's Included

### 📚 13 Comprehensive Rule Files

#### Poetry (4 files)
- **Sonnets** - Shakespearean sonnet form, rhyme scheme (ABAB CDCD EFEF GG), volta, themes
- **Narrative Poetry** - Venus and Adonis, The Rape of Lucrece, stanza forms, narrative techniques
- **Verse Forms** - Blank verse, rhymed couplets, prose vs. verse usage, metrical variations
- **Iambic Pentameter** - Meter, scansion, rhythm, substitutions, enjambment

#### Drama (4 files)
- **Tragedy** - Five-act structure, tragic hero, hamartia, peripeteia, anagnorisis, catharsis
- **Comedy** - Romantic comedy conventions, mistaken identity, disguise, comic relief
- **History Plays** - Historical accuracy vs. dramatic license, political themes, chorus
- **Scene Construction** - Opening scenes, dialogue balance, stage directions, pacing, climactic scenes

#### Language & Style (3 files)
- **Elizabethan English** - Pronouns (thou/thee/thy/thine), verb conjugations, archaic vocabulary, false friends
- **Rhetoric & Wordplay** - Metaphor, simile, puns, antithesis, alliteration, oxymoron, apostrophe
- **Vocabulary Guide** - Glossary of Elizabethan terms, pronunciation, etymology, graduated vocabulary lists

#### Character & Themes (2 files)
- **Character Development** - Complex characters, internal conflict, character arcs, foil characters, soliloquy
- **Thematic Elements** - Universal themes (love, power, ambition, revenge, fate vs. free will)

### 📝 11 Complete Examples

### Manual Installation

Copy the contents of this module to your project's `.augment/` folder.

## Quick Start

### Example 1: Write a Shakespearean Sonnet

```
Using the Shakespeare sonnets rules, write a sonnet about [TOPIC].

Requirements:
- Rhyme scheme: ABAB CDCD EFEF GG
- Meter: Iambic pentameter (10 syllables per line)
- Structure: 3 quatrains + 1 couplet
- Volta: Turn/shift in thought at line 9
- Theme: [THEME]
```

### Example 2: Create a Tragic Scene

```
Using the Shakespeare tragedy rules, write a scene for a tragic play.

Requirements:
- Five-act structure context: [ACT/SCENE]
- Tragic hero: [CHARACTER] with hamartia [FLAW]
- Include: Soliloquy revealing internal conflict
- Dramatic techniques: Metaphor, imagery, foreshadowing
- Theme: [THEME]
```

### Example 3: Write Elizabethan Dialogue

```
Using the Elizabethan English and rhetoric rules, write dialogue between [CHARACTER 1] and [CHARACTER 2].

Requirements:
- Use thou/thee/thy/thine appropriately
- Include: Puns, metaphors, antithesis
- Meter: Blank verse (unrhymed iambic pentameter)
- Tone: [FORMAL/INFORMAL/COMIC/TRAGIC]
- Purpose: [EXPOSITION/CONFLICT/RESOLUTION]
```

### Example 4: Develop a Complex Character

```
Using the character development rules, create a character profile for [CHARACTER].

Requirements:
- Internal conflict: [CONFLICT]
- Character arc: [POSITIVE/NEGATIVE/FLAT]
- Foil character: [FOIL]
- Soliloquy revealing: [MOTIVATION/DECISION/EMOTION]
- Thematic connection: [THEME]
```
